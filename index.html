How does this work?

Just add your Zip Code to see what events are near you. If would like to listen to the artist before purchasing the tickets, just enter the name of the artist and listen to a sample song.

Discover new artists and bands near you

How did we build this project?

We used a combination of two different API's Jambase and iTunes to create this project. Jambase allowed us to display events near the location the user would input in the Zip Code text box. Once we had a list of events displayed the user can search for a sample song by searching for the artist. The sample song the user will listen to will come from the iTunes API.
